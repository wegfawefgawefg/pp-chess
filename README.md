# pp-chess

its chess man.

me and geron tried to make it. started with a bit board, got stuck on some move behavior, ditched the bit board, and then it got abandoned.

Archive Note
------------

The original prototype was developed on September 30, 2023. The git history only has two original commits from that date, so this was a very short-lived chess experiment rather than a longer project.

The current repo was later cleaned up into a small playable CLI chess implementation with tests, but the original development burst was just that single September 30, 2023 session.
